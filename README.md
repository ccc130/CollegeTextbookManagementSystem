# CollegeTextbookManagementSystem
# 高校教材管理系统
## 系统分为系统管理员、教材管理员、学院管理员和任课教师四种角色。
### ①系统管理员:用户管理，教材类型管理，班级管理，任课教师管理
### ②教材管理员:出版社管理，教材库管理，教材的订购、入库、退回、领用管理，教材统计(按出版社、教材类型等)。
### ③学院管理员:审核本院教师的教材征订申请，查看教材征订情况，查看教材库，新增和管理本院所需教材。
### ④任课教师:查询教材库，提交教材征订申请，查看征订进度查。
### 【要求】教材的书号必须满足以ISBN开头，后跟10个数字的格式可选功能，教材统计信息的可视化图表展示与报表导出。
